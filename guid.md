

## Structure of the Code :

```
pizza_app/
│
├── main.py                  # App entry point, launches the window
├── requirements.txt         # customtkinter, python-escpos
│
├── ui/
│   ├── order_screen.py      # Main ordering interface
│   └── menu_panel.py        # Menu items and categories
│
├── db/
│   ├── database.py          # SQLite setup and queries
│   └── orders.db            # Auto-generated at first run
│
└── printer/
    └── printer.py           # ESC/POS print logic
```
